## Overview

Install local dotfiles and useful commands.

## Installation

```
git clone https://github.com/masa0x80/dotfiles.local.git ~/.dotfiles.local
cd ~/.dotfiles.local
make
```

Optional Steps:

Upgrade cask apps.

```sh
make brew-cask-upgrade
```
